INTRODUCTION OF HTML

What is HTML ?

HTML Stands for "Hyper Text Markup Language". By using HTML we can Create and Design like Static or Dynamic websites. Html is the building block of all websites. it tells the browser what to show and how to arrange content on the screen. here, Hyper Text means text with links to other web pages and Markup Language means it uses tags to define elements like headings, paragraph etc...

What are the tags in html ?

HTML provides basic structure of webpages by using some special codes. that codes are called as Tags. here two types of tags in html, 

    
1)Paired tags:
it is the combination of opentag and closetag Ex: h1 to h6, paragraph tags, div, span etc... 
    
2)Unpaired tags:
it consists of only opentag, close tag is not necessary. Ex: img, hr, br
